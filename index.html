Qu'est-ce que le projet M1 Devops

L'objectif du projet est de mettre en place une infrastructure en conteneur disposant d'un blog (sous wordpress), d'un plugin telegraf pour exposer la consommation du blog wordpress, un conteneur influxdb pour récolter la consommation du blog wordpress et l'afficher dans l'interface influxdb. Un autre conteneur comprenant l'usine logicielle (Gitlab) sera déployé pour maintenir le site web dans les évolutions futures (avec CI/CD).

Comment faire fonctionner le projet

L'ensemble de l'infrastructure est conteneurisé et déployable d'un coup via un docker-compose


Une fois le projet téléchargé : docker-compose up -d
